Input Types in HTML

form:
this element represents the beginning and end of your form in html
action:
the action attribute holds a value which is usually the URL (uniform resource locator) to the backend script which handles the form
method:
the method attrubute tells the form handler how to handle the form using the HTTP method described in the form.
